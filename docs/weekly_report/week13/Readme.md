## 최종으로 데모 영상 촬영 ##
  1. 실제로 와이파이를 연결할 수 없는 밖에서 드론을 원격으로 접속 한다.<br>
  2. 쓰레기통이 드론 와이파이에 접속하여 소켓 통신을 사용하여 드론과 쓰레기통이 통신할 수 있는 상황을 만든다.<br> 
  3. 쓰레기통에 쓰레기를 버려서 쓰레기통 센서값의 데이터를 드론과 주고받는다.<br>
